This tool creates simple interactive maps that you can display on your website. The intention is to help visualise and compare multiple data sets between regions.

This is a demonstration build, so please be kind to any bugs that cross your path!

Selecting a Map:

Start by loading a map from the provided templates. Map regions will automatically load into the table.

Navigation:

Use the zoom buttons to get a closer view.

Click and drag to move around the map.

Creating columns:

Click 'Add Column' to create a new column.

Click 'Delete Column' to remove column and any associated data.

The column name and associated data value will display when users click on a region.

Export:

Before exporting, set the desired zoom and location of your map. The exported HTML will use this position as default.

Click 'Export HTML Map' to generate an interactive map containing your own data. To use this in your own website, copy and paste the entire exported HTML in place.
